Math Properties and Methods

Properties

Property Explanation Code Returns
Math.PI Pi (π) Math.PI
Math.SQRT2 Square root of 2. Math.SQRT2

Methods

Method Explanation Code Returns
Math.abs(number) Absolute value of number. Math.abs(-12)
Math.ceil(number) number rounded up. Math.ceil(5.4)
Math.floor(number) number rounded down. Math.floor(5.6)
Math.max(numbers) Highest Number in numbers. Math.max(2, 5, 9, 3)
Math.min(numbers) Lowest Number in numbers. Math.min(2, 5, 9, 3)
Math.pow(number, power) number to the power of power. Math.pow(2, 5)
Math.round(number) Rounded number. Math.round(2.5)
Math.random() Random number between 0 and 1. Math.random()

var LOW = 1;
var HIGH = 10;
var RND1 = Math.random();
